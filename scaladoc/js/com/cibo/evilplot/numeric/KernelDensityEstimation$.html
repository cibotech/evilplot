  6.  def densityEstimate2D(data: Seq[Point], numPoints: (Int, Int) = (100, 100), xBounds: Option[Bounds] = None, yBounds: Option[Bounds] = None): GridData

    Perform a 2-dimensional kernel density estimation on data.

    Perform a 2-dimensional kernel density estimation on data.

    data

    A sequence of points, for which a density estimate is to be generated. "x" corresponds to columns of the resulting grid, "y" to rows.

    numPoints

    number of rows and number of columns of the resulting grid. Defaults to 100 x 100.

    xBounds

    desired boundaries of the grid in data coordinates. Defaults to min and max x value of data.

    yBounds

    desired boundaries of the grid in data coordinates. Defaults to min and max y value of data.

    returns

    A GridData object containing the 2D density estimation.
